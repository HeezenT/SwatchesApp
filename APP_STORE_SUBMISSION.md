# Shopify App Store Submission Draft

## App

Name: Deryan Variant Swatches

Primary language: English

Category suggestion: Store design

Short description:
Premium product variant swatches and product links for high-end Shopify storefronts.

App introduction:
Deryan Variant Swatches helps merchants replace basic variant selectors with premium, large-format swatches inspired by modern product configurators. Merchants can link products that represent different colors, customize the storefront appearance, add right-side helper text, and publish swatch data to Shopify metafields without editing theme code.

Feature list:
- Create product links for products that are the same item in different colors.
- Display large two-column swatch buttons with selected-state borders and color checkmarks.
- Customize active and inactive borders, backgrounds, text colors, radius, block height, and typography.
- Add helper text and optional info tooltips for variant rows.
- Sync product data and publish storefront swatch data through Shopify metafields.
- Works through a Shopify Online Store 2.0 theme app extension.

## Review Notes

This app does not request protected customer data. It stores Shopify shop session data, synced product identifiers, product handles, product titles, image URLs, product link groups, swatch settings, and appearance settings.

Mandatory compliance webhooks are implemented at:
https://deryan-variant-swatches-staging.onrender.com/webhooks

Configured compliance topics:
- customers/data_request
- customers/redact
- shop/redact

The app uses a theme app extension named `deryan-swatches`.
